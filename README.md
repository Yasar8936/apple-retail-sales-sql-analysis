## ЁЯУМ Project Overview

This project analyzes Apple retail sales data using **PostgreSQL** to extract meaningful business insights and answer real-world analytical questions. The dataset was imported into PostgreSQL using a Python ETL script built with **Pandas** and **SQLAlchemy**, followed by exploratory and business-focused SQL analysis.

The project demonstrates practical SQL skills including joins, Common Table Expressions (CTEs), window functions, aggregation, subqueries, ranking, date functions, and query optimization using indexes.

The objective of this project is to:

- Analyze Apple retail sales performance
- Evaluate warranty claim patterns

## ЁЯЪА How to Run

1. Create a PostgreSQL database.
2. Update the PostgreSQL credentials inside `python/load_data.py`.
3. Run the Python ETL script.

```bash
python load_data.py
```

4. Open pgAdmin.
5. Execute the SQL queries in:

```
sql/business_queries.sql
```
